# ps1-offline-hindi-voice-assistant
Offline, privacy-preserving Hindi voice assistant built for the Bharat AI-SoC Student Challenge (PS1). The system runs fully on-device without internet connectivity and focuses on campus-oriented use cases using edge AI.
## How to run (Windows)
1) Install Python 3.x
2) Install dependencies:
   pip install -r requirements.txt
3) Download Vosk Hindi model and keep it inside /models (as per project structure)
4) Install eSpeak-NG (Windows x64)
5) Run:
   cd src
   py app.py
